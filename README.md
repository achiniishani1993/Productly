# Productly
Recreate the design for Productly in Figma

This project is a simple landing page called Productly, created using HTML and CSS. It includes a navigation bar with sign in and sign up buttons, section with a call-to-action, feature cards, several content sections with images, and a footer with helpful links and a signup form. The layout is built mainly with Flexbox and uses the Google Poppins font for styling. To run the project, simply download the files and open the index.html file in your browser.
